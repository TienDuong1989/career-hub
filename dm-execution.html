Box 3
On-site Checklist
Per-page checklist. Run every time you publish or optimize a page.
Title tag + meta description (keyword in)
H1 unique · H2/H3 structured
Keyword in URL · first 100 words · alt text
Internal links (to pillar + related pages)
CTA clear · word count sufficient
Run per page · not one-time Google Drive ↗

Box 4
Technical SEO
Infrastructure-level. Set up once, monitor regularly. Different from on-site.
Core Web Vitals (LCP <2.5s · CLS · FID)
Crawlability · XML Sitemap · Robots.txt
301 Redirects · Broken links
Schema markup · HTTPS · Mobile-friendly
Index coverage (Google Search Console)
Audit quarterly · monitor monthly Google Drive ↗

Box 5
Content Cluster & Topic Authority
How to organize content so Google sees you as an authority on a topic — not just a blog.
1 Pillar page (broad topic, /software-dev)
Multiple Cluster pages (sub-topics)
Internal links: cluster → pillar (always)
Same topic cluster = same URL folder (/finance/*)
Structure first, then write Google Drive ↗
